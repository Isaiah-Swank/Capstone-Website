Envisioned Solution

We use a service oriented architecture to manage a web and mobile application that will allow for customizable and flexible meal prepping. We have four main pages: Calendar, Pantry, Recipes, and Profile which allow the user to manage ingredients and meals, create and select recipes, and customize a given week's meals.
